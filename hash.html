Documentation de hash.c/hash.h

hash.c/hash.h est un module qui implémente une version simple de table de hachage. Vous pouvez l'utiliser comme une « boîte noire ». Les éléments contenus dans les tables sont des chaînes de caractères sans valeur associée.

Structure HASH : elle contient les informations relatives à une table de hachage. Vous n'accédez pas directement à son contenu, vous y accédez uniquement au travers des fonctions fournies.

int hash_init(HASH *hash, int size) : cette fonction initialise une table de hachage pointée par hash avec une taille de size éléments. Le fonctionnement sera optimal si vous choisissez une taille de table de 1,5 à 2 fois la taille maximum des éléments (uniques) à stocker. Elle retourne SUCCESS en cas de réussite et FAILURE en cas d'échec (SUCCESS et FAILURE sont définis dans hash.h).

int hash_add(char *term, HASH *hash) : cette fonction ajoute la chaîne de caractères term à la table pointée par hash. Elle retourne SUCCESS en cas de réussite et FAILURE en cas d'échec.

int hash_entries(HASH *hash) : cette fonction retourne le nombre d'entrées uniques dans la table pointée par hash.

int hash_sort(HASH *hash) : cette fonction trie par ordre alphabétique les entrées uniques de la table pointée par hash. Elle retourne SUCCESS en cas de réussite et FAILURE en cas d'échec.

char **hash_list(HASH *hash) : cette fonction retourne la liste des entrées uniques dans la table pointée par hash. Elle retourne un pointeur nul en cas d'échec.

int hash_test(char *term, HASH *hash) : cette fonction retourne 1 si la chaîne de caractères term est présente dans la table pointée par hash et 0 sinon.

int hash_num(char *term, HASH *hash) : cette fonction retourne l'indice de la chaîne de caractères term si elle est présente dans la table pointée par hash et -1 sinon.

Vous n'avez normalement pas besoin des autres fonctions de ce module.
